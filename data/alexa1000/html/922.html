OpenX Hosting Features

A single OpenX server can serve over 10,000,000 impressions per day while a nine-server cluster handles more than 400,000,000 impressions daily.

In order to install OpenX on a multi-server cluster we utilize the distributed statistics plugin. We configure MySQL replication on the forward facing web servers as slaves with an admin database server that serves as the master. We then install load balancers in front of the web servers to ensure equal distribution of load, and optionally integrate a CDN (content delivery network) to serve the images. The result is a highly available OpenX installation that is built for superb ad delivery and easy scalability.
